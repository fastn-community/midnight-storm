Creating posts list page

It's very common to display a list of top/latest posts as a blog home page. Our midnight-storm blog feature helps you to create your favorite blog posts list home page.

It's very simple to create your top/latest featured posts using -- ds.posts which can be added inside -- ds.page.

Let's learn how to create it.
Example of how to use -- ds.posts
-- import: fastn-community.github.io/midnight-storm/blog-articles/category-1/sample-post-1

-- ds.page:

-- ds.posts:

-- ds.featured-post:
post-data: $sample-post.meta

-- ds.image-first:
post-data: $sample-post.meta

-- ds.image-in-between:
post-data: $sample-post.meta

-- ds.without-image:
post-data: $sample-post.meta

-- ds.image-first:
post-data: $sample-post.meta

-- ds.image-in-between:
post-data: $sample-post.meta

-- end: ds.posts

-- end: ds.page
post-list.ftd

ftd
